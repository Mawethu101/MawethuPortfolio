For this project, I created a website based on the NASA API, transforming raw space data into visually compelling data art and interactive graphs. This experience allowed me to explore the intersection of technology, data, and creativity, as I learned how to extract, analyze, and repurpose real-time NASA data into engaging digital visuals.

Serving as the web developer, UI/UX designer, data analyst, and data extractor, I designed an intuitive user interface while ensuring seamless data integration and visualization. This project not only strengthened my technical skills but also deepened my understanding of how data can be transformed into an artistic and interactive experience.
